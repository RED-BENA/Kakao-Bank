Q 해외송금 신청시 OTP가 필요한가요?

해외송금 신청금액이 원화기준 1천만원 초과일 경우 OTP인증이 필요합니다.
※당행 OTP 가 없으신 경우, 타행 OTP 이용등록 후 사용하실 수 있습니다.

Q 인도로 해외송금 시 은행코드는 무엇을 입력해야하나요?

인도 송금시 은행코드는 반드시 IFSC Code 영문/숫자 11자리를 넣으셔야 합니다.

IFSC Code is required by Kakaobank when sending INR to the India.

Q 호주로 해외송금 시 은행코드는 무엇인가요?

호주 송금시 은행코드는 BSB Number 숫자 6자리 입니다.

BSB Number is required by Kakaobank when sending AUD to the Australia.

Q 뉴질랜드 해외송금 시 은행코드는 무엇인가요?

뉴질랜드 송금시 은행코드는 National Clearing Code 숫자 6자리 입니다.
뉴질랜드 계좌번호(15자리 혹은 16자리)는 다음과 같이 구성되어 있습니다.
  National Clearing Code 6자리 + 계좌번호 9자리 혹은 10자리
    ex) 계좌번호 : 011848 1234567890 (16자리)
    - 은행코드(National Clearing code) : 011848
    - 송금 시 입력 계좌번호: 1234567890 (10자리)
  계좌번호가 9자리이면 끝 2자리 앞에 숫자 ‘0’을 넣어서 10자리를 만들어서 입력합니다.
  ex ) 계좌번호 : 011848 123456789 (15자리)
    - 은행코드(National Clearing code) : 011848     - 송금 시 입력 계좌번호: 1234567089 (숫자 ‘0’을 추가하여 10자리로 입력)


National Clearing Code is required by Kakaobank when sending NZD to the New Zealand.

Q [전월세보증금 대출] 코픽스금리는 무엇인가요?

COFIX(Cost of Funds Index)는 국내 8개 은행(정보제공은행)들이 제공한 자금조달 관련 정보를 기초로 하여 산출되는 자금조달비용지수로서, "잔액기준 COFIX", "신규취급액기준 COFIX" 및 "단기COFIX"로 구분 공시됩니다.
전월세보증금 대출은 전국은행엽합회에 2영업일 전 공시된 "신규취급액 COFIX"를 기준금리로 하여 대출금리에 적용합니다.

Q [전월세보증금 대출] 이미 잔금을 납입한 경우에도 전월세보증금 대출이 가능한가요?

이용중이신 대출상품이 한국주택금융공사의 보증서를 활용한 대출이라면 해당 대출을 상환 하신 후, 카카오뱅크의 전월세보증금 대출을 신청하셔야합니다.
(한국주택금융공사는 부부합산 1건의 전세자금보증을 지원하고 있습니다.)

Q [전월세보증금 대출] 전월세보증금 대출을 받았는데요, 보증료가 출금되었어요. 보증료가 뭔가요?

한국주택금융공사는 원활한 주택공급을 위하여 한국주택금융공사법에 따라 전월세자금대출을 이용하는 고객에게 보증서를 발급하여 주고 이에 대한 보증료를 수납하고 있습니다.
카카오뱅크는 고객님으로부터 보증료 현가할인을 받으실 수 있도록 일시납 수납하여 공사에 전달하고 있습니다.
또한, 대출의 중도상환 시에는 잔존일수와 금액에 대하여 보증료를 환출해드립니다.

Q [전월세보증금 대출] 대출신청은 언제까지 해야하나요?

임대차계약서 상의 잔금일 기준으로 1개월 전부터 15일 전까지 신청이 가능하십니다.
다만, 한도와 금리조회의 경우는 잔금일 기준 3개월 전부터 이용하실 수 있습니다.

Q [전월세보증금 대출] 결혼을 앞둔 신혼부부는 대출 진행이 불가능한가요?

예비부부이시더라도 카카오뱅크 전월세보증금 대출 신청이 가능하십니다. 다만, 혼인신고 전이시라면 신혼부부가 아닌 "미혼"자격으로 대출신청을 하셔야합니다.

Q [전월세보증금 대출] 임대차계약서를 분실했습니다. 어떻게 제출하나요?

임대차계약서를 분실하신 경우 계약한 기관에서 아래 해당서류를 증빙하는 서류를 발급받아 제출해 주시기 바랍니다.
1) 공공임대사업자(LH) : 계약사실확인
2) 공공임대사업자(SH) : 원본대조필이 확인된 임차계약
3) 부동산중개업소 : 임대차계약서 사본(부동산보관용), 확정일자 신고내역서

Q [전월세보증금 대출] 확정일자를 꼭 받아야하나요?

임대차계약서 상 확정일자는 필수입니다.
단 임대인이 공공임대사업자(LH, SH 등)인 경우에는 확정일자를 생략할 수 있습니다.

Q [전월세보증금 대출] 대출이 가능한 주택은 어떻게 되나요?

등기사항전부증명서(등기부등본) 상 구분 등기된 주택(아파트, 연립주택, 다세대주택, 주거용 오피스텔 등)
단, 다음 중 하나에 해당하는 주택은 불가
1) 임차대상 주택의 소유자와 임대인이 일치하지 않는 경우
2) 등기사항전부증명서(등기부등본) 상 소유권에 (가)압류 등 권리침해가 있는 경우
3) 미등기 또는 무허가건물
4) 신청인의 직계존비속이 소유한 주택

Q [전월세보증금 대출] 소유자가 법인인 경우도 가능한가요?

소유자가 법인인 경우에는 해당 법인이 공공임대사업자인 경우에만 가능합니다

Q [전월세보증금 대출] 임대차계약서상 쌍방계약(공인중개사 없애 개인간의 거래)도 가능한가요?

부동산중개업소를 통해 임대차계약을 체결한 경우에만 대출신청이 가능합니다.
단, 임대인이 공공임대사업자인 경우 또는 임대차계약 갱신인 경우에는 임대인-임차인 쌍방계약도 가능합니다.

Q [전월세보증금 대출] 대출금 수령일자를 임대차 계약서상 잔금일과 다르게 할 수 있나요?

임대차계약서 상 잔금납부일(입주예정일) 전/후 7일 이내에 임대인 명의의 계좌로 대출금 송금 가능합니다

Q [전월세보증금 대출] 임대차보증금은 금액에 상관없이 가능한가요?

수도권(서울/경기/인천)은 4억원 이하, 수도권 이외는 2억원 이하인 경우 가능합니다.
최소 보증금은 1천만원 이상인 경우 가능합니다

Q [전월세보증금 대출] 서류제출이 가능한 시간은 언제인가요?

카카오뱅크 전월세대출은 모바일, PC 공인인증서 확인 및 이미지 첨부로 제출이 가능합니다.
1) 평일 : 08시부터 22시까지
2) 토요일 : 08시 ~19시까지
3) 일요일과 공휴일 : 불가

Q [전월세보증금 대출] 공동명의로 계약한 주택입니다. 전월세보증금 대출이 가능한가요?

카카오뱅크에서는 계약자(임차인)가 공동명의인 경우 대출이 불가합니다.

Q [전월세보증금 대출] 신혼가구 (혼인 신고일 기준 5년 미만)인 경우 우대사항이 있나요?

한국주택금융공사에 납부하는 보증료에 대한 할인과 보증한도(대출한도)에 우대가 있습니다.

Q [전월세보증금 대출] 임대차계약시 저 대신 대리인이 계약서를 작성했습니다. 대출이 가능한가요?

임대차계약서 상 본인(임차인,계약자)에 대한 대리행위가 있는 경우에는 대출이 불가합니다.
